# 项目简介

## 核心功能
- 基于YOLOv8的香蕉成熟度检测，支持6类分类（新鲜成熟、新鲜未成熟、过熟、成熟、腐烂、未成熟）
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供图形用户界面，可实时显示目标位置、目标总数和置信度信息
- 支持自定义置信度和IOU阈值调整
- 提供模型选择功能，可切换不同检测模型
- 支持批量检测功能，可查看所有检测结果
- 提供检测结果保存功能

## 适用场景
- 香蕉种植园的成熟度评估
- 香蕉加工厂的分级检测
- 农产品质量控制系统
- 农业科研与教学演示
- 智能农业检测系统开发参考

## 技术特点
- 使用PyCharm+Anaconda或VSCode+Anaconda作为开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 内置10000张标注数据集，涵盖全部6类香蕉成熟度状态
- 提供完整源码、训练好的模型、数据集和运行教程
- 支持实时预览和批量处理功能
